Introductions

MarQ Admin is a fully responsive admin template based on bootstrap 4 version. It is fully responsive built using SASS preprocessor, HTML5, CSS3 and jQuery plugins. MarQ have also Material Design components, which you can use in your application. It works natively on mobile devices, making it an accessible way of managing your own website from wherever you are.

MarQ Admin comes with many example pages with many ready to use components and easy to customize. MarQ offers multiple theme options, you can chose as per your choice.

This documentation will guide you through installing the template and exploring the various components that are bundled with the template.

Dependencies
  • Bootstrap 4
  • Nodejs
  • Sass
  • jQuery v3.3.1

Installation

Installation Guide

After download package you can find pre build app folder in marq/html/light or marq/html/dark packages. Choose your desire template and you just need to copy and paste app folder in your desire location.

Note: we suggest you to please make your one own css file & one js files and add that in your page, so whenever the update of Admin Pressadmin comes it does not affect your code.

Prerequisite For Html Template

  • For Html template move to your desire template light or dark in your terminal and execute following commands.
  • Node.js and NPM : You can download Node.js from https://nodejs.org. NPM comes bundled with Node.js.
  • Install bower locally : You can install bower in local machine by executing from your terminal npm install -g bower .
  • Install gulp locally : You can install gulp in local machine by executing from your terminal npm install gulp -g .
  • Install dependency : After installing Node, npm and gulp run npm install command from the root of your project directory into terminal. It will install all the necessary dependencies for the application using package.json file. After that run bower install from terminal to install remaining dependency from bower.json file.
  • Build : Run gulp command from project directory to build the project. All require files are compile and place in /app directory.
  • Just copy and paste app folder in your location. That's it!!!

Prerequisite For Angular 6+ Template

  • For Angular template move to Marqangular directory in your terminal and execute following commands.
  • Node.js and NPM : You can download Node.js from https://nodejs.org. NPM comes bundled with Node.js.
  • Angular Cli : You can install angular-cli using following command install -g @angular/cli
  • Install bower locally : You can install bower in local machine by executing from your terminal npm install -g bower .
  • Install gulp locally : You can install gulp in local machine by executing from your terminal npm install gulp -g .
  • Install dependency : After installing Node, npm and gulp run npm install command from the root of your project directory into terminal. It will install all the necessary dependencies for the application using package.json file. After that run bower install from terminal to install remaining dependency from bower.json file.
  • Run gulp command from project directory to compile all scss files and generate necessary js in assets folder.
  • Run Project: Run ng serve command from your terminal in project directory, and navigate to http://localhost:4200/ into your browser.
  • Build Project: For build project execute command ng build from terminal in project directory. So dist folder is generated in project directory.
